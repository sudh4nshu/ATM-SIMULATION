**💳 Project 1: ATM Simulation**
A functional banking application that mimics a real-world ATM interface.

**Key Features:** Balance Inquiry, Cash Deposit, Cash Withdrawal, and Transaction Statements.

**Technical Logic:**
* Uses Global Variables to track account balance and history across multiple operations.
Implements Input Validation to ensure users cannot withdraw more money than they possess.

* Uses an infinite while loop with a break command for a seamless user experience.
